# crawlers
## description
写过的一些爬虫汇总，仅供交流，爬虫有害网络环境，谨慎批量使用。

## structure
```python
├── ...
├── code                        # 代码部分
│   └── bilibili                ## b站相关
│       ├── danmaku_history.py  ### 历史弹幕爬虫
│       └── danmaku_recent.py   ### 最近弹幕爬虫
└── data                        # 样例数据部分，结构同代码部分
```

## usage
- 进入虚拟环境
`pipenv shell`
- 安装相关的包
`pipenv install`
